2. 아래의 명령어를 설명하시오(뭐 하는 건가).
-javac
[.java파일을 컴파일하는 명령어]

**javac는 .java 파일을 컴파일 하는 명령어**로 javac [classname.java](http://classname.java) 명령어를 실행하면 
classnaem.class 라는 컴파일된 .class 파일(JVM에서 실행되는 바이트 코드)이 생성된다. 
이렇게하면 컴퓨터(CPU)가 이해할 수 있게 되고,  컴퓨터에서 실행이 가능해진다

-java
[실행]
java 명령어는 javac 명령을 통해 컴파일한 **.class 파일을 실행하는 명령어.
java classname(.class생략) 명령을 사용해 작성한 소스코드의 내용이 컴퓨터(CPU)에서 실행된다. (실제로는 JVM에서 실행.)


3. 컴파일이란 무엇인가요?
[컴퓨터가 이해할 수 있는 0101덩어리로 바꾸는 것]

컴파일이란 프로그래머가 작성한 소스 코드 파일을 컴퓨터가 이해할 수있는 기계어로 번역해 주는 과정. 

Java, C, C++ 등의 대부분의 언어는 인간이 이해할 수 있는 언어로 작성된다. 하지만 컴퓨터는 0과1로 이루어진 이진수만 이해할 수 있기 때문에 프로그램을 실행하기 위해서는 *번역하는 과정이 필요하다.
그것을 컴파일 이라고 하며 *컴파일을 해주는 프로그램을 컴파일러 라고 한다.
