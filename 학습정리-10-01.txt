*8. int num; 을 메모리 그리고 설명해보시오.
 정수형 데이터타입 (자료형)으로 4byte의 메모리공간을 할당받는 
변수 num을 선언한다

int num;은 int(정수)형 타입의 num(변수 이름) 변수를 선언한다는 것으로,
이것은 int 데이터 타입의 4byte만큼을 이 변수를 위해 메모리 공간을 할당한다는 것
즉, num 변수의 메모리 공간이 4byte 만큼 확보된다는 것을 의미

10. JDK란 무엇이며, 어디서 다운로드 받으며, OS별로 버전이 있는 까닭은?
  JDK는Java Development Kit의 약자로 자바개발키드이다 java 환경에서 돌아가는 프로그램을 개발하는데 필요한 툴을 모아놓은 소프트웨어 패키지이다.
 JDK를 설치하면 자바프로그램을 동작시키기 위한 실행환경인  JRE와 JVM이 모두 설치된다

 JAVA 파일은 운영체제에 독립적으로 작동하는데 .JAVA 파일로 작성한 자바 파일이 .class파일로 컴파일 되어 실행되는데
운영체제에 상관없이 실행되는것을 독립적이라고 이야기 한다

이렇게 독립적으로 파일이 실행되기 위해서 실행되는 환경을 갖추어야하는데, JRE가 제대로 갖춰져 있어야하며 이 때문에
JDK를 설치할 때 운영체제별로 다르게 설치하는 것이다



11. JVM 무엇의 약자이며,이란 무엇인가?
 JAVA Virtual Machine의 약자
 .class 파일을 OS 에 맞는 각 운영체제가 알아들을 수 있는 코드로 변환시켜줌
